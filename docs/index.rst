pytopoviz documentation
=======================

Welcome to the documentation for ``pytopoviz``. This project extends
the Python ``topotoolbox`` API with visualization-first utilities powered by
PyVista and quick CPU-accelerated helpers using Numba. Unlike the core package,
this extension does not rely on ``libtopotoolbox``.
